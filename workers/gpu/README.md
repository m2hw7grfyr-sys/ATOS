# ATOS GPU Worker

This worker is the minimal Main-to-GPU loop for ATOS.

Flow:

1. Send heartbeat to ATOS Main.
2. Lease one queued GPU generation task.
3. Mark the task as started.
4. Call Ollama `/api/generate` in non-stream mode.
5. Complete or fail the task.

It does not start or stop Vast instances, open browsers, call TGE, or run Playwright.

## Configuration

Copy the example file:

```bash
cp /workspace/atos-gpu-worker/config/gpu-worker.env.example /workspace/config/gpu-worker.env
```

Required values:

- `MAIN_URL`: ATOS Main URL, for example `http://host.docker.internal:8080`
- `GPU_WORKER_API_KEY`: dedicated GPU worker bearer token generated by Main
- `OLLAMA_URL`: usually `http://127.0.0.1:11434`
- `MODEL_NAME`: for example `llama3.1:8b`

Do not commit the real `gpu-worker.env`.

## Status Check

```bash
python3 /workspace/atos-gpu-worker/main.py \
  --config /workspace/config/gpu-worker.env \
  --status
```

## Run Once

```bash
python3 /workspace/atos-gpu-worker/main.py \
  --config /workspace/config/gpu-worker.env \
  --once
```

## Run Forever

```bash
/workspace/atos-gpu-worker/scripts/run_worker.sh
```

Supervisor can run this command with `autostart=false`.
