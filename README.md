# ATOS

ATOS 项目文档、原型、数据库、API 与源代码的统一仓库。

## 目录

- `docs/`：产品、架构、实施和决策文档
  - `specifications/`：正式规范与冻结版本
  - `drafts/`：评审稿与工作草稿
  - `diagrams/`：Mermaid、draw.io 和架构图
  - `ui-wireframes/`：页面原型与交互线框
  - `adr/`：架构决策记录
  - `templates/`：可复用文档模板
- `assets/`：图片、架构图和 UI 资源
- `prototype/`：产品原型
- `sql/`：数据库脚本
- `api/`：API 定义与服务
- `src/`：应用源代码
